# Agent 内容更新手册

> 此目录存放所有内容数据 JSON 文件
> 更新内容时只需修改此目录下的文件，无需改动代码

## 文件说明

| 文件 | 说明 |
|------|------|
| `manifest.json` | 全局版本号，修改后客户端会自动刷新 |
| `math.json` | 数学课课程列表 |
| `science.json` | 科学可视化场景列表 |
| `social-scenes.json` | 社交训练场景目录 |
| `gallery.json` | 童画廊画作列表 |
| `neimen.json` | 内功养生法卡片列表 |

## 更新流程

1. 修改对应 JSON 文件
2. 更新 `manifest.json` 中的版本号（如 `2026-05-30-v1` → `2026-05-30-v2`）
3. 客户端下次访问时会自动检测版本变化并刷新

## 注意事项

- 不要删除必填字段（id、title 等）
- 新增内容时确保 id 唯一
- 图片路径使用 `/images/xxx/` 格式
